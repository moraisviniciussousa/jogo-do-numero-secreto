# 🎮 Jogo do Número Secreto

Projeto prático desenvolvido para consolidar fundamentos de lógica de programação, manipulação de estruturas condicionais e interatividade.

## 🚀 Tecnologias Utilizadas
- **HTML5:** Estruturação da página.
- **CSS3:** Estilização e design responsivo.
- **JavaScript:** Lógica do jogo, geração de números pseudo-aleatórios e controle de fluxo.

## 🧠 Conceitos Aplicados
- Variáveis e tipos de dados.
- Estruturas de repetição (`while`) e condicionais (`if/else`).
- Funções e escopo de código.
- Manipulação básica do DOM (Document Object Model).

---
*Desenvolvido durante a formação de Lógica de Programação na Alura.*
